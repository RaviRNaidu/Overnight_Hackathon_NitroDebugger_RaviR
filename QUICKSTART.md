# Quick Start Guide

##  Start the Backend Server

Open PowerShell and run:

```powershell
cd C:\Users\arunk\OneDrive\Desktop\farmer-portal\backend
pip install -r requirements.txt
python main.py
```

##  Open the Frontend

Open another PowerShell window and run:

```powershell
cd C:\Users\arunk\OneDrive\Desktop\farmer-portal\frontend
Start-Process index.html
```

##  Test the Application

1. **Submit an Application**:
   - Click "Application Form" in navbar
   - Fill all fields and submit
   - Note the Application ID

2. **Track Your Application**:
   - Click "Application Tracker"
   - Enter Application ID and Mobile Number
   - View status

3. **Department Login**:
   - Click "Department Login"
   - Use credentials: `admin` / `admin123`
   - Successful login will show confirmation
